# Wild Code School

Dépôt personnel regroupant les quêtes, exercices et projets réalisés durant la
formation **Data / IA** à la Wild Code School.

## Objet

Espace de travail pédagogique : chaque sous-dossier correspond à un thème ou un
projet abordé en formation (notebooks Jupyter, scripts Python, mini-applications
Streamlit). Le contenu évolue au fil des quêtes — la structure du dépôt n'est
donc pas figée et change régulièrement.

## Environnement

- **Langage** : Python 3
- **Format principal** : notebooks Jupyter (`.ipynb`) et scripts Python (`.py`)
- **Dépendances** : un `requirements.txt` à la racine recense l'ensemble des
  bibliothèques utilisées à travers les différents exercices (data science,
  scraping, NLP, visualisation, interfaces web, etc.)

## Statut

Dépôt vivant, mis à jour au rythme des quêtes. Les commits suivent une convention
simple (`add: quete <nom>`) pour tracer l'avancement.
